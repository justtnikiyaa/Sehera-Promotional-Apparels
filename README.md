# Sehera Promotional Apparels Website

Frontend-only company website built with React, Vite, Tailwind CSS, React Router, and Framer Motion.

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```

3. Build production files

```bash
npm run build
```

4. Preview production build

```bash
npm run preview
```
